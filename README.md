# FraudLens — Financial Crime & Behavioral Intelligence Suite

> End-to-end fraud analytics project covering transaction-level anomaly detection,
> behavioral risk profiling, and operational forensics — built in Tableau.

---

## Overview

FraudLens is a portfolio analytics project simulating a real-world sportsbook/fintech
fraud operations environment. It transforms raw transactional data into two
production-grade Tableau dashboards that surface fraud patterns, customer risk tiers,
and forensic drill-downs for investigation teams.

The project demonstrates the full analyst workflow: raw data → cleaning → modelling
→ dashboard design → executive storytelling.

## How to Use

1. Download `Fraud_Small.csv` and the `.twbx` workbook
2. Open the `.twbx` file in Tableau Desktop (v2022.1+)
3. The packaged workbook includes the data source — no reconnection needed
4. Use the dashboard filters to explore fraud patterns by date, risk tier, and channel
